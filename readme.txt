The Zica Portfolio Posts plugin is a WordPress plugin that will let you add a new custom post type 'Portfolio'.

== Description ==

The Zica Portfolio Posts plugin adds a new custom post type 'Portfolio' to a WordPress instance. This plugin is spplements the Zica and Zica Lite WordPress themes which utilize WordPress's custom post type functionality.


== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/zica-portfolio' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. You will now see a new category of posts - 'Portfolio' will appear to which you can add content.
